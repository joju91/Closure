\# Efterplan — Product, Company \& App Master Plan



Efterplan is a unified planning system for building the company, the product, and the app with clarity, determinism, and zero friction.  

This repository contains structured documentation extracted from the planning sheets.



\## Structure



\- `dashboard.md` — High‑level overview and guiding principles  

\- `manifest.md` — Full checklist of product principles  

\- `company-plan.md` — Business, operations, and distribution plan  

\- `app-plan.md` — UX, flows, and feature definitions  

\- `roadmap.md` — Milestones, timeline, and dependencies  



\## Core Philosophy



\- \*\*Mobile‑first\*\* — Everything is designed primarily for mobile use  

\- \*\*Zero friction\*\* — The user should never wonder “what now?”  

\- \*\*One decision per step\*\* — Cognitive load is always minimized  

\- \*\*No AI in runtime\*\* — Deterministic logic in the core flow  

\- \*\*Copy is action\*\* — Clear, directive language (“Do this now”)
